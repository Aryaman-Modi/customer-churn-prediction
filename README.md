# Customer Churn Prediction using Machine Learning

An end-to-end machine learning project to predict customer churn using supervised learning techniques. The project analyzes customer demographics, service usage, and subscription details to identify customers at high risk of leaving, enabling proactive customer retention strategies.

## Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. This project develops a predictive model that classifies whether a customer is likely to churn based on historical customer data.

## Dataset

- **Records:** 12,055 customers
- **Target Variable:** Churn
- **Features:** 20 customer demographic and service-related attributes
